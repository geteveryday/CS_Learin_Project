# 解读

```reStructuredText
-  VL引脚 --设置对比度，值越小对比度越高

-  RC引脚 -- 高电平 -》操作数据  --低电平-》操作指令

- R/W引脚  --高电平为读，低电平为写

- E引脚 --低到高时使能-》引脚上的值有效，高到低关闭-》引脚上的值无效

- 7-14 传输电信号的引脚

- 背光源的正负极，为屏幕提供光源，与屏幕成像原理有关。供电用
```

## RS R/W解析

| RS   | R/W  | 操作   |
| ---- | ---- | ------ |
| 0    | 0    | 写命令 |
| 1    | 1    | 读数据 |
| 0    | 1    | 读命令 |
| 1    | 0    | 写数据 |
